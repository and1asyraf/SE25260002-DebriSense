5.5. Summary of Important Findings

The system testing and evaluation phase was conducted to verify that the DebriSense platform has fulfilled all its functional requirements. Therefore, testing was carried out comprehensively at three levels: unit testing, integration testing, and system testing.

During the unit testing phase, all critical components were tested individually to confirm they functioned as expected. This included the core predictive logic: the computation of the Debris Risk Index (DRI) based on environmental parameters, and the algorithm that calculates debris payload and waste breakdown based on land-use baselines. Additional components such as interactive map rendering, user authentication, and data input validation were also thoroughly tested. Test cases were strictly designed to cover normal operations, boundary limits, and invalid inputs. All unit test cases were executed successfully without any major defects.

Integration testing focused on ensuring smooth interactions between the platform's internal modules and external services. The testing verified that data flowed accurately and securely between the frontend user interface and the backend relational database. Key integration points, including the successful parsing of external weather API data for real-time DRI updates and the seamless connection to the Gemini API for the interactive chatbot, were rigorously tested and performed reliably.

System testing evaluated the complete end-to-end functionality of DebriSense by simulating real-world scenarios. This included full workflows where NGO users interact with the map, toggle personalized watchlists, submit empirical hotspot cleanup reports, and receive automated background alerts when a river's DRI reaches 'High' or 'Critical' severity. Additionally, administrator workflows for verifying NGO accounts and approving new location requests were validated. The application was also tested for responsiveness and usability, indicating a smooth, intuitive, and cohesive user experience across all modules.

In conclusion, the testing and evaluation phase was successful across all levels. The high success rate of the test cases indicates that the system is both stable and highly reliable. DebriSense is ready for real-world deployment, equipped with the necessary predictive capabilities and empirical reporting tools to support NGOs and administrators in proactive river debris management and targeted cleanup logistics.
